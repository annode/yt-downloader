YouTube Downloader Web starten
==============================

1. ZIP entpacken.
2. start_youtube_downloader_web.bat doppelklicken.
3. Im Browser oeffnen:
   http://127.0.0.1:8766

Im LAN erreichst du die App ueber die lokale IP des Rechners, z.B.:
http://192.168.178.119:8766

Hinweise:
- Python muss installiert sein.
- yt-dlp muss installiert sein.
- Fuer Transkripte muss openai-whisper installiert sein.
- FFmpeg muss fuer Audio/Video-Konvertierung verfuegbar sein.
